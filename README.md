# Academic Portfolio & Asset Management Template

A responsive, high-performance static portfolio template optimized for researchers, academics, and engineers. Built with vanilla HTML/CSS and deployed using Netlify Edge Functions for ultra-fast asset optimization and delivery.

## Features
- **Edge-Powered Assets:** Uses Netlify Edge Functions to compress and sync core data models dynamically.
- **Clean Architecture:** Lightweight footprint with zero unnecessary dependencies, ensuring low latency.
- **SEO Optimized:** Structured semantic HTML designed for academic indexers.

## Project Structure
- `/netlify/edge-functions/`: Core script handlers for dynamic asset mapping.
- `index.html`: Main landing page showcasing academic achievements and project telemetry.
